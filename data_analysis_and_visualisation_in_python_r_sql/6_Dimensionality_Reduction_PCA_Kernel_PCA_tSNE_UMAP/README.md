# HW6: Dimensionality Reduction: PCA, Kernel PCA, t-SNE, and UMAP

This assignment studies how four dimensionality reduction methods behave on the Fashion-MNIST subset provided in the starter notebook:

- PCA
- Kernel PCA with an RBF kernel
- t-SNE
- UMAP

## What this experiment does

The notebook:

- loads the Fashion-MNIST data from the starter code
- uses the assignment sampling scheme to create a balanced subset of 2000 points
- standardizes the data before building embeddings
- generates baseline 2D embeddings for PCA, Kernel PCA, t-SNE, and UMAP
- performs hyperparameter sweeps for t-SNE and UMAP
- saves the comparison figures into `figs_results/`
- selects the best visual t-SNE and UMAP settings
- creates a final `2x2` comparison plot
- measures runtime for all four final methods

## Final chosen settings

From the completed notebook, the final visual choices are:

- t-SNE: `perplexity=50`, `learning_rate=100`
- UMAP: `n_neighbors=30`, `min_dist=0.1`
