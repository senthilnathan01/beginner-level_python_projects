# HW8: Tree-Based Methods

This repository contains the completed notebook implementation, exported figures, dataset file used in the analysis, and the final report PDF for HW8 in MA 5755.

## Higher-Level Goal

This assignment studies how tree-based methods learn structure, how sensitive that structure is to perturbations, and how ensembles and Bayesian tree models change the tradeoff between interpretability and predictive performance.

The flow of the assignment is intentional:

1. A single regression tree is used to expose the learned splits and subgroup rules directly.
2. Variants of the tree are compared to show instability and the effect of model depth.
3. A classification tree is examined through terminal leaves, predicted probabilities, and calibration behavior.
4. Random Forest and Gradient Boosting are compared against a single tree on the same housing split.
5. BART is used to extend the tree framework to a probabilistic model with predictive uncertainty.

The broader aim is to understand not only which method predicts better, but how the model design changes the partition of feature space, the stability of predictions, and the kind of interpretation each method supports.

## What Was Done

The notebook:

- uses the Ames Housing data for regression and the Breast Cancer Wisconsin data for classification,
- keeps a common train/test split for the regression comparisons with `random_state = 42`,
- uses a stratified train/test split for classification with the same random state,
- builds a full regression tree for structure discovery and a shallow companion tree for interpretable subgroup summaries,
- compares deep, shallow, and perturbed trees on the same test set,
- studies classification leaves through selected observations, leaf compositions, and Brier scores,
- compares a single tree, Random Forest, and Gradient Boosting using the same housing split,
- fits a BART model on a reduced top-feature subset to keep Bayesian sampling tractable in the available environment.

## Main Results

### Regression Tree and Linear Benchmark

- The dominant root split is `Overall Qual <= 7.5`.
- The top levels are driven mainly by `Overall Qual` and `Gr Liv Area`.
- Representative high-end subgroup: `Overall Qual > 8.5` and `Gr Liv Area > 2229`, with `n = 36` and mean `SalePrice = 439713.61`.
- Representative lower-priced subgroup: `Overall Qual <= 4.5` and `Total Bsmt SF <= 799.5`, with `n = 90` and mean `SalePrice = 86017.77`.
- On this split, the regression tree test RMSE is about `33246.80`, while linear regression gives about `20984.32`.

### Tree Instability

- Deep tree: depth `26`, leaves `1560`, test RMSE `33246.80`.
- Shallow tree: depth `4`, leaves `15`, test RMSE `35722.61`.
- Perturbed deep tree: depth `24`, leaves `1421`, test RMSE `33527.31`.
- The top split pattern stays stable, but lower-level structure and point predictions change noticeably.

### Classification Tree

- Logistic regression Brier score is `0.0284`.
- The selected high-confidence error falls in a large pure leaf, showing that tree overconfidence is not limited to tiny mixed leaves.

## Theoretical Takeaways

- A single tree is easiest to interpret because it exposes concrete if-then rules, but its fitted structure is unstable.
- Small changes in data can leave the main split intact while still changing many lower-level branches and test predictions.
- Tree probabilities come from empirical leaf proportions, so they can be coarse and overconfident.
- Random Forest improves mostly by averaging many unstable trees and reducing variance.
- Gradient Boosting improves by sequentially correcting residual structure and can achieve lower bias.
- BART distributes signal across many weak trees and contributes posterior predictive uncertainty that the standard ensemble methods do not provide.

## Overall Result

The main conclusion is that Gradient Boosting is the strongest predictive method among the models tested on this split of the Ames Housing data. Random Forest is also much stronger than a single tree, while BART sits between the single-tree and ensemble baselines in point prediction quality. The single tree remains the clearest model for subgroup interpretation, but it is the least stable. BART is the most informative when uncertainty matters because it provides a posterior distribution rather than only a point prediction.
